# esako-prueba

Réplica del sitio web de ESAKO (estática, lista para cPanel).

## Páginas
- `index.html` — Inicio
- `Nosotros.dc.html` — Empresa
- `Servicio.dc.html` — Servicio
- `Soluciones.dc.html` — Soluciones
- `Oportunidades.dc.html` — Oportunidades
- `Tienda.dc.html` — Tienda
- `support.js` — framework de render + extras (menú móvil, botón flotante WhatsApp, ES/ING)
- `.htaccess` — caché y compresión para Apache/cPanel

## Despliegue
Subir todos los archivos a la raíz del dominio (`public_html` o la carpeta del subdominio).
